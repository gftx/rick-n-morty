## How to start / Запуск приложения

- Clone repo / Клонировать репозиторий

```js
git clone https://github.com/gftx/rick-n-morty
```

- Install dependencies / Установить зависимости

```bash
yarn
# или
npm i
```


- Launch project / Запустить проект

```bash
yarn start
# или
npm start
```
